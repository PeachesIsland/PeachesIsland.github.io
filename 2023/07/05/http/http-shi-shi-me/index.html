http是什么？

http 是超文本传输协议 ，是一个应用层的协议

主要特点：

  1. 支持客户/服务器模式
  2. 简单快速
  3. 灵活
  4. 无连接，http1.0 使用短链接，http1.1（目前主流） 起默认使用长连接（keep-alive）。
  5. 无状态

http请求结构有哪些？

  1. 请求行（Request Line），包含请求方法、请求的URL和HTTP协议的版本。
  2. 请求头（Request Headers）包含一系列的键值对，用于描述请求的各种属性和要求。常见的请求头包括：
  • Host：指定服务器的域名或IP地址。
  • User-Agent：标识发送请求的用户代理（通常是浏览器）。
  • Accept：指定可接受的响应内容的MIME类型。
  • Content-Type：指定请求体的MIME类型（仅适用于POST、PUT等包含请求体的方法）。
  • Authorization：用于身份验证的凭证信息
  1. 空行：请求头和请求体之间必须有一个空行。
  2. 请求体（Request Body）：可选项，包含发送给服务器的数据，通常用于POST、PUT等需要提交数据的请求方法

http响应报文有哪些？

  1. 状态行（Status Line） 包含HTTP协议版本、状态码和对应的状态信息，示例：HTTP/1.1 200 OK
  2. 响应头（Response Headers）
  3. 空行：响应头和响应体之间必须有一个空行。
  4. 响应体（Response Body）

常见http状态码有哪些？

200：ok，正常返回信息

400：客户端请求语法错误

403：重定向

404：客户端请求的资源不存在

500：服务器错误

503：服务器错误
